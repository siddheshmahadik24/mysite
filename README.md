# Siddhesh Mahadik — Personal Website

Personal portfolio site built with React, TypeScript, and Vite. Live at **[www.siddhesh-mahadik.com](https://www.siddhesh-mahadik.com)**.

## Tech Stack

- **React 18** + **TypeScript**
- **Vite** — build tool & dev server
- **Tailwind CSS** — styling
- **Framer Motion** — animations & lightbox transitions
- **Sharp** — image optimisation
- **Railway** — hosting & deployment
- **Cloudflare** — DNS & CDN

## Running Locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
```

## Deployment

Pushes to `master` auto-deploy to Railway via GitHub integration.
